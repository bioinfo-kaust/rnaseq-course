# rnaseq-course

Static HTML + slide deck materials for a one-week RNA-seq data analysis course at KAUST: https://bioinfo-kaust.github.io/rnaseq-course

## How to view

This repo has no build step. Open the files in your browser, or serve the folder locally.

```bash
cd /path/to/rnaseq-course
```

Then open homepage: `index.html`

## Repo structure (high level)

- `index.html` and `html/*.html`: course pages and lab instructions
- `css/style.css`: shared stylesheet
- `slides/`: PPTX slide decks
- `html/`: related lab and course web pages
