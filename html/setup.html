Python Environment on Ibex (for Jupyter Notebooks)

Labs 5 and 7 include optional Python Jupyter notebooks for exploratory analysis and visualization. The steps below install Miniconda and mamba on Ibex, then create a dedicated rnaseq-python environment with all required packages.

Run these commands on Ibex, not on your local machine.

Connect to Ibex first (ssh username@ilogin.ibex.kaust.edu.sa), then follow the steps below in your home directory.

Step 1 — Install Miniconda

# Download the Miniconda installer for Linux (64-bit)
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh \
    -O ~/miniconda3_install.sh

# Run the installer in batch mode (no interactive prompts)
bash ~/miniconda3_install.sh -b -p ~/miniconda3

# Activate conda for your current shell session
source ~/miniconda3/bin/activate

# Initialise conda so it loads automatically in future sessions
conda init bash

# Reload your shell config
source ~/.bashrc

# Verify
conda --version

Step 2 — Install mamba

Mamba is a faster drop-in replacement for the conda solver. Install it once into the base environment:

conda install -n base -c conda-forge mamba -y

# Verify
mamba --version

Step 3 — Create the rnaseq-python environment

This environment includes JupyterLab and all Python packages used in the course notebooks (data handling, PCA, plotting, and differential expression analysis):

mamba create -n rnaseq-python -c conda-forge -c bioconda \
    python=3.11 \
    jupyterlab \
    ipykernel \
    pandas \
    numpy \
    matplotlib \
    seaborn \
    scikit-learn \
    scipy \
    pydeseq2 \
    adjusttext \
    -y

Step 4 — Activate and verify

conda activate rnaseq-python

# Check that key packages are installed and importable
python -c "import pandas, numpy, matplotlib, seaborn, sklearn, pydeseq2; print('All packages OK')"

# Confirm Jupyter Lab is available
jupyter lab --version
