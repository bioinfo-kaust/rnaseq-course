Prerequisites

  • An active Ibex HPC account at KAUST
  • A laptop with a terminal application (macOS Terminal, Linux shell, or Windows with WSL2)
  • Basic familiarity with the Linux command line

Software & Tools on Personal Computer

Command-line Tools (via conda/mamba)

Tip We recommend installing tools via mamba (a faster drop-in replacement for conda). 
# Create a dedicated conda environment
mamba create -n rnaseq -c bioconda -c conda-forge \
    fastqc fastp trimmomatic multiqc \
    star hisat2 samtools \
    subread salmon kallisto \
    rseqc qualimap sra-tools \
    -y

conda activate rnaseq

R Packages (Bioconductor)

if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install(c(
    "DESeq2",
    "edgeR",
    "tximport",
    "tximeta",
    "clusterProfiler",
    "org.Hs.eg.db",
    "fgsea",
    "pheatmap",
    "ComplexHeatmap",
    "EnhancedVolcano",
    "apeglm"
))

install.packages(c("ggplot2", "dplyr", "tidyr", "ggrepel"))

Connecting to Ibex

# Connect to Ibex via SSH
ssh your_username@ibex.kaust.edu.sa

Use pre-installed software and tools

# Search for a tool and load if found
module avail toolName
module load toolName
