Lab Overview

Lab 1

Public Data Access & SRA Download

Browse GEO and SRA to find RNA-seq datasets. Download FASTQ files using sra-tools (prefetch + fasterq-dump) and ENA browser. Organise a project directory structure.

Lab 2

Quality Control

Run FastQC and MultiQC on raw RNA-seq reads. Interpret QC metrics and identify potential issues.

Coming soon
Lab 3

Read Trimming

Trim adapters and low-quality bases using fastp. Compare pre- and post-trimming QC reports.

Coming soon
Lab 4

Alignment with STAR

Build a STAR genome index and align trimmed reads. Assess alignment statistics and visualize in IGV.

Coming soon
Lab 5

Read Counting

Quantify gene expression using featureCounts. Explore the count matrix and normalization strategies.

Coming soon
Lab 6

Salmon Quantification

Build a Salmon index and perform quasi-mapping-based quantification. Import results with tximport.

Coming soon
Lab 7

Introduction to nf-core/rnaseq

Install Nextflow and nf-core tools on Ibex. Understand pipeline structure, profiles, and configuration. Run the pipeline on a small test dataset and explore the output directory.

Coming soon
Lab 8

Configuring & Running nf-core/rnaseq

Build a samplesheet, set pipeline parameters (aligner, genome, strandedness), run with the Ibex SLURM profile, and monitor progress. Troubleshoot common issues.

Coming soon
Lab 9

Interpreting nf-core/rnaseq Output

Navigate the pipeline output: MultiQC report, STAR/Salmon results, bigWig files, and QC flags. Understand which outputs feed into downstream DE analysis.

Coming soon
Lab 10

Differential Expression with DESeq2

Perform DE analysis in R with DESeq2. Filter, shrink, and interpret DEG results.

Coming soon
Lab 11

Visualization

Generate PCA plots, volcano plots, MA plots, and heatmaps to visualize expression patterns.

Coming soon
Lab 12

Functional Enrichment

Run GO and KEGG enrichment analysis with clusterProfiler. Visualize and interpret results.

Coming soon
Lab 13

nf-core/differentialabundance

Configure and run the nf-core/differentialabundance pipeline on a count matrix. Define contrasts, review the automated DESeq2/edgeR outputs, and explore the interactive Shiny report.

Coming soon
Lab 14

End-to-end Real Dataset Capstone

Apply the full workflow to a real public RNA-seq dataset: download from SRA, run nf-core/rnaseq, import counts, perform DE analysis, visualize results, and produce a final report.
