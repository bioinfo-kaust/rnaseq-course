RNA-seq Data Analysis Course

From Raw Reads to Biological Insights — A Hands-on Training Program

Dates 5-9 April 2026 | KAUST | Building 9 | Room 2120

Learning Outcomes

📚
RNA-seq Principles

Understand the biological and technical principles of RNA sequencing, library preparation, and experimental design

💻
Read Processing & QC

Perform quality control, trimming, and pre-processing of raw RNA-seq reads using industry-standard tools

🧬
Alignment & Quantification

Align reads to reference genomes and transcriptomes; quantify gene and transcript expression

📄
Differential Expression

Identify differentially expressed genes using DESeq2 and edgeR and interpret statistical results

🌟
Visualization & Reporting

Produce publication-quality plots, heatmaps, volcano plots, and comprehensive analysis reports

🎓
Functional Analysis

Perform gene ontology enrichment and pathway analysis to translate DEG lists into biological insight

🔨
Production Workflows with nf-core

Run a fully reproducible, scalable RNA-seq analysis using the nf-core/rnaseq pipeline on real datasets on HPC

Program Overview

Module 1

Introduction, Experimental Design & Data Access

RNA-seq biology and technology overview, experimental design principles, sequencing strategies (bulk, strand-specific, paired-end), HPC/Linux refresher, and downloading public RNA-seq data from SRA/ENA.

View Module 1 →
Module 2

Read QC & Pre-processing

Raw data quality assessment with FastQC and MultiQC, adapter trimming and quality filtering with fastp and Trimmomatic.

View Module 2 →
Module 3

Alignment & Quantification

Reference-based alignment with STAR and HISAT2, alignment QC with RSeQC and Qualimap, and read counting with featureCounts and HTSeq.

View Module 3 →
Module 4

Quasi-mapping & Transcript Quantification

Fast transcript-level quantification with Salmon and kallisto, understanding TPM/FPKM/RPKM, and tximport for downstream analysis.

View Module 4 →
Module 5

nf-core/rnaseq Workflow

Introduction to Nextflow and nf-core. Configure and run the nf-core/rnaseq pipeline on Ibex — covering samplesheet setup, pipeline parameters, profiles, and understanding the full output structure.

View Module 5 →
Module 6

Differential Expression Analysis

Exploratory data analysis, normalization, statistical modelling with DESeq2 and edgeR, and interpreting DE results including LFC shrinkage.

View Module 6 →
Module 7

Visualization & Functional Analysis

PCA plots, volcano plots, heatmaps with pheatmap/ComplexHeatmap, gene ontology enrichment with clusterProfiler, and pathway analysis with fgsea.

View Module 7 →
Module 8

nf-core/differentialabundance

End-to-end automated DE analysis using the nf-core/differentialabundance pipeline — from count matrices to interactive Shiny reports, covering configuration, contrasts, and output interpretation.

View Module 8 →
Module 9

Real-world Analysis Capstone

End-to-end analysis of a real public RNA-seq dataset: running nf-core/rnaseq, interpreting MultiQC reports, importing Salmon counts, performing DE analysis, and reporting results.

View Module 9 →
